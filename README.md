# BSIS 3A – Flutter Quiz Starter Application

A Flutter-based quiz starter application developed as part of the **BSIS 3rd Year Mobile Application Development coursework**.  
This project demonstrates proper Flutter environment setup, GitHub usage, and a basic quiz flow implemented using **`setState` only**, in compliance with the instructor’s requirements.

---

## 📌 Project Overview

### Title
**Flutter Quiz Starter App (BSIS 3A)**

### Description
This application is a simple quiz system built using Flutter. It features a start screen, quiz interaction, and result display. The app focuses on fundamental Flutter concepts such as widget layout, state management using `setState`, timers, and responsive UI design.

### Project Status
✅ **Completed** (Academic Requirement)
